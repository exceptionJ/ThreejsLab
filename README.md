# ThreejsDemos

使用 Threejs 实现的3D效果demo。

记录自己学习的一个过程。

所有项目均使用了2个辅助库：
*stats.js性能监视器来监视3d性能，详见各个demo左上角（它的介绍在https://github.com/mrdoob/stats.js ）
*GUI库( dat.gui.js )，详见各个demo右上角，用户可进行控制操作（https://github.com/dataarts/dat.gui）
